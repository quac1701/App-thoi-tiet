APP sử dụng API để hiển thị thông tin về thời tiết tại những địa điểm nhất định, có hỗ trợ tìm kiếm bằng giọng nói.
app sử dụng 2 API:
- Api trả về tọa độ địa lý của địa điểm được người dùng đánh lên thanh tìm kiếm.
- Api dùng tọa độ của địa điểm để trả về các chỉ số thời tiết realtime

App có sử dụng thư viện webkitSpeechRecognition để nhận diện giọng nói.
